Устаревшее ключевое слово "var"

var - это устаревшее объявление переменной.

Для «var» не существует блочной области видимости

Видимость переменной объявленной при помощи var ограничена лишь функцией, во всех остальных вариантах, или почти во всех, var является глобальной переменной: 

    
function sayHi() {
  var phrase = "Привет"; // локальная переменная, "var" вместо "let"

  alert(phrase); // Привет
}

sayHi();

alert(phrase); // Ошибка: phrase не определена
Но: 
    
if (true) {
  var test = true; // используем var вместо let
}

alert(test); // true, переменная существует вне блока if
То же самое касается циклов, блоков кода.

«var» допускает повторное объявление

            
var user = "Пётр";

var user; // ничего не делает, переменная объявлена раньше
// ...нет ошибки

alert(user); // Пётр

«var» обрабатываются в начале запуска функции

    
function sayHi() {
  phrase = "Привет";

  alert(phrase);

  var phrase;
}
sayHi();
Но тут будет ошибка: 
    
function sayHi() {
  var phrase; // объявление переменной срабатывает вначале...

  alert(phrase); // undefined

  phrase = "Привет"; // ...присвоение - в момент, когда исполнится данная строка кода.
}

sayHi();

IIFE

Расшифровывается как: «Immediately-invoked function expressions» (сокращенно IIFE).

Способ как обойти глобальную видимость переменной объявленой var.

Внимание: объявлять переменные и мужественно обходить такое объявление не надо. Скорее всего такое встретится тебе примерно никогда

В целом способ следующий: 

        
(function() {

  var message = "Привет";

  alert(message); // Привет

})();
По сути метод разбития кода на блоки, ничуть не лучше глобального var. Несколько способов потанцевать на граблях ещё: 
    
// Способы создания IIFE

(function() {
  alert("Круглые скобки вокруг функции");
})();

(function() {
  alert("Круглые скобки вокруг всего выражения");
}());

!function() {
  alert("Выражение начинается с логического оператора НЕ");
}();

+function() {
  alert("Выражение начинается с унарного плюса");
}();

Итого - объявлять переменные var нахуй не надо - забудь.
