BigInt

Пишет новая возможность в 2023 году.

BigInt – это специальный числовой тип, который предоставляет возможность работать с целыми числами произвольной длины. Варианты создания: 

    
const bigint = 1234567890123456789012345678901234567890n;

const sameBigint = BigInt("1234567890123456789012345678901234567890");

const bigintFromNumber = BigInt(10); // то же самое, что и 10n
И тут автор упускает момент, что например 1n - обозначение BigInt 
    
alert(typeof 1n)

Математические операторы

Можно использовать как обычные числа, но нельзя миксовать с обычными числами: 

    
alert(1n + 2n); // 3

alert(5n / 2n); // 2
alert(1n + 2); // Error: Cannot mix BigInt and other types
Для совместного использования надо конвертировать в BigInt или Number: 
    
let bigint = 1n;
let number = 2;

// конвертируем number в bigint
alert(bigint + BigInt(number)); // 3

// конвертируем `bigint` в number
alert(Number(bigint) + number); // 3
Второе преобразование опасно, потому что может не хватить знаков. Так же к BigInt нельзя применить унарный оператор +

Операции сравнения

Можно сравнивать как и обычные числа, можно сравнивать с обычными числами, но с нюансом: 

    
alert( 2n > 1n ); // true

alert( 2n > 1 ); // true
alert( 1 == 1n ); // true

alert( 1 === 1n ); // false

Логические операции

В if или любом другом логическом операторе bigint число ведёт себя как обычное число. К примеру, в if bigint 0n преобразуется в false, другие значения преобразуются в true: Логические операторы ||, && и другие также работают с bigint числами как с обычными числами: 

    
if (0n) {
  // никогда не выполнится
}

alert( 1n || 2 ); // 1

alert( 0n || 2 ); // 2
