Intl: интернационализация в JavaScript

Для решения проблем со строками, датами и числами была создана эта реализация в javascript. Раздел я побегу коротенько в виду редкой востребованности, точно не первостепенной.

Основные объекты

Intl.Collator Умеет правильно сравнивать и сортировать строки.

Intl.DateTimeFormat Умеет форматировать дату и время в соответствии с нужным языком.

Intl.NumberFormat Умеет форматировать числа в соответствии с нужным языком.

Локаль

Локаль описывается строкой из трёх компонентов, которые разделяются дефисом:

  1. Код языка.
  2. Код способа записи.
  3. Код страны.
например zh-Hans-CN – китайский язык (zh), записываемый упрощённой иероглифической письменностью (Hans), используемый в Китае. Если локаль не указана или undefined – берётся локаль по умолчанию, установленная в окружении (браузере).

Подбор локали

localeMatcher – вспомогательная настройка, которую тоже можно везде указать, она определяет способ подбора локали, если желаемая недоступна. По умолчанию стоит "best fit".

Строки, Intl.Collator

                
let collator = new Intl.Collator([locales, [options]])
Можно использовать для сортировки: 
    
let result = collator.compare(str1, str2);
//Например:
let collator = new Intl.Collator();

alert( "ёжик" > "яблоко" ); // true (ёжик больше, что неверно)
alert( collator.compare("ёжик", "яблоко") ); // -1 (ёжик меньше, верно)

Даты, Intl.DateTimeFormat

                
// создание
let formatter = new Intl.DateTimeFormat([locales, [options]])
Например: 

    
let date = new Date(2014, 11, 31, 12, 30, 0);

let formatter1 = new Intl.DateTimeFormat("ru");
alert( formatter1.format(date) ); // 31.12.2014

let formatter2 = new Intl.DateTimeFormat("en-US");
alert( formatter2.format(date) ); // 12/31/2014
Далее там ещё есть настройки на время, время и дату в национальном формате.

Числа, Intl.NumberFormat

                
let formatter = new Intl.NumberFormat([locales[, options]]);

formatter.format(number); // форматирование
С пару десятков различных настроек, просто приведу пару примеров: 
    
let formatter = new Intl.NumberFormat("ru");
alert( formatter.format(1234567890.123) ); // 1 234 567 890,123
let formatter = new Intl.NumberFormat("ru", {
  style: "currency",
  currency: "GBP"
});

alert( formatter.format(1234.5) ); // 1 234,5 £

Методы в Date, String, Number

                
let str = "ёжик";

alert( str.localeCompare("яблоко", "ru") ); // -1
Форматирует дату в соответствии с локалью, например: 
    
let date = new Date(2014, 11, 31, 12, 0);

alert( date.toLocaleString("ru", { year: 'numeric', month: 'long' }) ); // Декабрь 2014
И ещё ряд методов.

Задача

            
let animals = ["тигр", "ёж", "енот", "ехидна", "АИСТ", "ЯК"];

animals.sort((a, b) => a.localeCompare(b))

alert( animals ); // АИСТ,ёж,енот,ехидна,тигр,ЯК

Но автор предлагает по иному: 

    
let collator = new Intl.Collator();
animals.sort((a, b) => collator.compare(a, b));
Мой вариант работает и корректен 100%.
