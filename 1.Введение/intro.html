Введение в JavaScript

Что особенного в js и чего с ним можно достичь.

Что такое JavaScript?

JS это изначально интерпретируемый язык распорстраняемый месте с html страничками. Сейчас можно выполняться отдельно от html на специальных интерпретаторах node, bun и т.д. Внутри браузера работает на движках по типу V8.

Что может JavaScript в браузере?

Может например:

  • Добавлять html-код на страничку.
  • Обрабатывать события возникающие в результате действий пользователя. В том числе и создавать интерактивность веб-ресурса.
  • Работать с сетью.
  • Устанавливать куки
  • Работать с локальным хранилищем на стороне пользователя в браузере.

Чего НЕ может JavaScript в браузере?

Все ограничения JS напрямую связаны с безопасностью пользователя, для того, чтобы минимизировать риск злоупотреблений:

  • JS на веб-странице не может работать с файлами пользователя напрямую, только через инициацию таких действий пользователями. С камерой и микрофоном аналогично.
  • Окна и вкладки не могут общаться друг с другом, в том числе и разные ресурсы не могут без явного разрешения общаться друг с другом. См "политика одного источника".
  • Веб-страница не может обратиться к другому ресурсу кроме того, с которого она пришла. Если пользователь не задал иное ключами запуска.
