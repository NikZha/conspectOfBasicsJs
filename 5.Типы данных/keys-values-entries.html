Object.keys, values, entries

Как и у Map, Set, Srray у объекта тоже есть эти методы, но они немного отличаются:

  • Object.keys(obj) – возвращает массив ключей.
  • Object.values(obj) – возвращает массив значений.
  • Object.entries(obj) – возвращает массив пар [ключ, значение].
Следует обратить внимание что синтаксис именно такой Object.keys(obj) потому что объект может сам реализовывать методы с таким называнием.
Перебор значений в цикле реализуется следующим образом: 
            
let user = {
  name: "John",
  age: 30
};

// перебор значений
for (let value of Object.values(user)) {
  alert(value); // John, затем 30
}
Так же можно использовать цикл: 
            
for (let [key, value] of Object.entries(obj)) console.log(`key: ${key} value: ${value}`)

Трансформации объекта

У объектов нет множества методов, которые есть в массивах, например map, filter и других. Если мы хотели бы их применить, то можно использовать Object.entries с последующим вызовом Object.fromEntries:

  1. Вызов Object.entries(obj) возвращает массив пар ключ/значение для obj.
  2. На нём вызываем методы массива, например, map.
  3. Используем Object.fromEntries(array) на результате, чтобы преобразовать его обратно в объект.
Наример, есть цены которые нужно удвоить: 
            
let prices = {
  banana: 1,
  orange: 2,
  meat: 4,
};

let doublePrices = Object.fromEntries(
  // преобразовать в массив, затем map, затем fromEntries (обратно в объект)
  Object.entries(prices).map(([key, value]) => [key, value * 2])
)
