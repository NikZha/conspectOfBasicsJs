Строки

Внутренний формат для строк — всегда UTF-16, вне зависимости от кодировки страницы.

Кавычки

В JS есть несколько видов кавычек:

  • Одинарные и двойные кавычки никак ничем не отличаются друг от друга. Правда лучше использовать что-то одно в коде. Вопрос больше стиля кода.
  • Обратные кавычки позволяют писать в несколько строк, оборачивать переменные в конструкции вида: ${...}. Что местами сокращает код.

Спецсимволы

Это символы которые невидимы на строке, такие как первод каретки, табуляция. Или кавычки. Список:

Символ Описание
\n Перевод строки
\r В текстовых файлах Windows для перевода строки используется комбинация символов \r\n, а на других ОС это просто \n. Это так по историческим причинам, ПО под Windows обычно понимает и просто \n.
\', \", \` Кавычки
\\ Обратный слеш
\t Знак табуляции
\b, \f, \v Backspace, Form Feed и Vertical Tab — оставлены для обратной совместимости, сейчас не используются.

Длина строки

Строки имеют свойство length это именно свойство к которому обращаются напрямую как к свойству. Например (на двойной клик по коду):

            
alert( `My\n`.length ); // 3

Доступ к символам

Делается или в стиле массива в [] квадратных скобках или при помощи функции .at() в скобках указывают номер символа с нуля до length-1 соотвественно с первого до последнего символа.

Так же доступ к символам возможен в цикле for...of:

            
for (let char of "Hola") {
    alert(char); 
}

Строки неизменяемы

Впрямую строку изменить нельзя.

Изменение регистра

Методы toLowerCase() и toUpperCase() меняют регистр символов. Так же это можно провернуть и с отдельным символом. Но надо понимать, что строки неизменяемы, то что возвращается этой функцией - это отдельная другая строка.


                
let str = 'Hola'
alert(str.toUpperCase())
alert(str[0].toLowerCase())
alert(str)

Как мы видим исходная строка не изменилась.

Поиск подстроки

str.indexOf(substr, pos)

Он ищет подстроку substr в строке str, начиная с позиции pos, и возвращает позицию, на которой располагается совпадение, либо -1 при отсутствии совпадений.

includes, startsWith, endsWith

Более современный метод str.includes(substr, pos) возвращает true, если в строке str есть подстрока substr, либо false, если нет.

Методы str.startsWith и str.endsWith проверяют, соответственно, начинается ли и заканчивается ли строка определённой строкой.

Получение подстроки

В JavaScript есть 3 метода для получения подстроки: substring, substr и slice.

str.slice(start [, end])
Возвращает часть строки от start до (не включая) end.

str.substring(start [, end])
Возвращает часть строки между start и end (не включая) end.
Это — почти то же, что и slice, но можно задавать start больше end.
Если start больше end, то метод substring сработает так, как если бы аргументы были поменяны местами.

str.substr(start [, length])
Возвращает часть строки от start длины length.

Сравнение строк

Можно сравнивать строки впрямую по типу >:

            
alert( 'a' > 'Z' ); // true

Но надо быть готовым к неправильному поведению, когда окажется что те символы которые не ASCII будут больше тех которые ASCII.

Чтобы избежать подобного поведения используют str.localeCompare(str2) вызов этого метода возвращает:

  • Отрицательное число, если str меньше str2.
  • Положительное число, если str больше str2.
  • 0, если строки равны.
