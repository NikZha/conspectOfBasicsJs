Методы объекта, "this"

Чтобы соотвествовать предметной области у объекта есть методы, которые могут совершать действия над внутренним содержанием объекта.

Примеры методов

В учебнике слишком беззубый пример, я предлагаю свой, основанный на фабричном методе создания объекта и методов в нём и сразу на ES6:

            
function makeSquare(x) {
let obj = {
        x,
        sqrt() { return this.x * this.x },
        cube() { return this.x * this.x * this.x },
    }
    return obj
}
let two = makeSquare(2)
alert(two.sqrt())
alert(two.cube())

Хоть автор и будет рассматривать this. позже, но я сделаю пример таким каким писал бы его в реальном коде.

Ключевое слово «this» в методах

Предыдущий пример показывает использование этого ключевого слова для доступа к внутреннему содержанию объекта.

«this» не является фиксированным

По факту фукнция (не стрелочная) тоже является объектом и по этому у неё есть this, если она находит у себя то свойство то работает с ним, если нет, то работает с внешним окружением. Если не находит то будет или undefined, а в "use strict" будет ошибка.

У стрелочных функций нет своего «this»

Стрелочная функция не создаёт собственный контекст this. Вместо этого она использует this из внешнего (окружающего) кода. Это удобно для колбэков, но делает стрелочные функции непригодными для методов объекта или конструкторов.

Задачи

Задача 2 Создайте калькулятор

Код объекта калькулятора по задаче 
 
                
let calculator = {
    x: 0,
    y: 0,
    read() {
        this.x = Number(prompt('Enter x:'))
        this.y = Number(prompt('Enter y:'))
    },
    sum() {
        return this.x + this.y;
    },
    mul() {
        return this.x * this.y
    }
};

Дипсик сообщает, что инициация x и y в объекте не требуется. Переменные внутри объекта создаст read(). Естественно надо будет делать проверку на ошибки в других методах объекта.

Задача 3 Цепь вызовов

Чтобы работала цепочка вызовов:

                
ladder.up().up().down().showStep().down().showStep(); // показывает 1 затем 0

Я изменил объект таким способом:


                
let ladder = {
    step: 0,
    up() {
        this.step++;
        return this
    },
    down() {
        this.step--;
        return this
    },
    showStep: function () { // показывает текущую ступеньку
        alert(this.step);
        return this
    }
};

Ключевое тут return this.
