Методы объекта, "this"

Чтобы соотвествовать предметной области у объекта есть методы, которые могут совершать действия над внутренним содержанием объекта.

Примеры методов

В учебнике слишком беззубый пример, я предлагаю свой, основанный на фабричном методе создания объекта и методов в нём и сразу на ES6:

            
function makeSquare(x) {
let obj = {
        x,
        sqrt() { return this.x * this.x },
        cube() { return this.x * this.x * this.x },
    }
    return obj
}
let two = makeSquare(2)
alert(two.sqrt())
alert(two.cube())

Хоть автор и будет рассматривать this. позже, но я сделаю пример таким каким писал бы его в реальном коде.
