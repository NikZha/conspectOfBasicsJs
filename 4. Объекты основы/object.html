Объекты

Ещё один тип данных которых объединяет разные типы данных в виде свойств объекта. Ситаксис создания:


let user = new Object(); // синтаксис "конструктор объекта"
let user = {};  // синтаксис "литерал объекта"

Литералы и свойства

При использовании литерального синтаксиса можно создавать напрямую свойства объекта:


let user = {     // объект
  name: "John",  // под ключом "name" хранится значение "John"
  age: 30        // под ключом "age" хранится значение 30
};

Вызываются свойства путём обращения к ним через точку:


// получаем свойства объекта:
alert( user.name ); // John
alert( user.age ); // 30

Можно добавлять новые свойства так же через точку:


user.isAdmin = true;

Свойства так же легко удаляются:


delete user.age;

Если свойство состоит из нескольких слов оно должно быть закавычено:

        
let user = {
  name: "John",
  age: 30,
  "likes birds": true  // имя свойства из нескольких слов должно быть в кавычках
};

Последнее слово объекта тоже может оканчиваться запятой, что упрощает добавление свойств:

            
let user = {
  name: "John",
  age: 30,
}

Объект созданый при помощи const может быть изменён:

            
const user = {
  name: "John"
};

user.name = "Pete"; // (*)

alert(user.name); // Pete

const тут служит для того, чтобы переменная не переопределялась.
