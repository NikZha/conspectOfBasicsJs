Условное ветвление: if, '?'

В разделе будет рассмотрены инструкции if, else, ?.

Инструкция «if»

Проверка условия вида:


let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', '');

if (year == 2015) alert( 'Вы правы!' );

При соотвествии могут быть несколько инструкций которые должны быть заключены в скобки {}.

Преобразование к логическому типу

  • Число 0, пустая строка "", null, undefined и NaN становятся false. Из-за этого их называют «ложными» («falsy») значениями.
  • Остальные значения становятся true, поэтому их называют «правдивыми» («truthy»).

Блок «else»

Инструкция if может содержать необязательный блок «else» («иначе»). Он выполняется, когда условие ложно.

Несколько условий: «else if»

Иногда нужно проверить несколько вариантов условия. Для этого используется блок else if.

Условный оператор „?“

По другому тенарный оператор:


let result = условие ? значение1 : значение2;

Если условие верно, то выбирается значение 1, если false то значение 2. Лучше не использовать, читаемо слабо. Я вспомнил эту языковую конструкцию у же на стадии чтения топика, на стадии создания оглавления даже не осознал что это про тенарный оператор. Поэтому только за деньги.

Несколько операторов „?“

Последовательность операторов вопросительного знака ? позволяет вернуть значение, которое зависит от более чем одного условия. Например: 


let age = prompt('Возраст?', 18);

let message = (age < 3) ? 'Здравствуй, малыш!' :
  (age < 18) ? 'Привет!' :
  (age < 100) ? 'Здравствуйте!' :
  'Какой необычный возраст!';

alert( message );

Нетрадиционное использование „?“

Иногда оператор «вопросительный знак» ? используется в качестве замены if:


let company = prompt('Какая компания создала JavaScript?', '');

(company == 'Netscape') ?
   alert('Верно!') : alert('Неправильно.');
