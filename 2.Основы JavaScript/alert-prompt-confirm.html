Взаимодействие: alert, prompt, confirm

Это функции для взаимодействия с пользователем. В настоящее время используются крайне редко - все продпочитают самописные модельные окна внутри браузера.

alert

alert показывает сообщение и ждёт, пока пользователь нажмёт кнопку «ОК». Окно модальное.

prompt


result = prompt(title, [default]);

Где:

  • title - заголовок окна.
  • default - необязательный параметр, который появится в строке ввода.

confirm


result = confirm(question);

Окошко с сообщением question. И двумя кнопками Ок и Отмена. Возвращает соотвественно true или false
