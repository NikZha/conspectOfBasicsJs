Типы данных

JS - динамически типизированный язык, что означает что одна и та же переменная может попеременно представлять строки, числа, объекты и т.д. Что бывает весьма удобно.

Число

Тип данных для числа по умолчанию Number можно приводить другие типы чисел нему как Number(x). Скорее является double64. То есть это и целочисленные операции и с плавающей точкой сразу: 


                let a = 2
                let b = 4
                let c = 5
                alert(b / a)
                alert(c / a)

Существует ряд специальных значений:

  • Infinity - бесконечность. Может быть -Infinity.
  • NaN - не число, обозначает ошибку. Вернуть обратно к числу нельзя, проверяют при помощи ф-ции isNaN в блоках типа if.

BigInt

// символ "n" в конце означает, что это BigInt
const bigInt = 1234567890123456789012345678901234567890n;

Строка

Строка string в JavaScript должна быть заключена в кавычки.


let str = "Привет";
let str2 = 'Одинарные кавычки тоже подойдут';
let phrase = `Обратные кавычки позволяют встраивать переменные ${str}`;
let name = "Иван";

// Вставим переменную
alert( `Привет, ${name}!` ); // Привет, Иван! ОБРАТИТЬ ВНИМАНИЕ НА ЗНАК ДОЛЛАРА

// Вставим выражение
alert( `результат: ${1 + 2}` ); // результат: 3

Другие кавычки не дают встраивать переменные.

В JavaScript null не является «ссылкой на несуществующий объект» или «нулевым указателем», как в некоторых других языках. Это просто специальное значение, которое представляет собой «ничего», «пусто» или «значение неизвестно». Специальное значение undefined также стоит особняком. Оно формирует тип из самого себя так же, как и null. Оно означает, что «значение не было присвоено».

Булевый (логический) тип

Булевый тип (boolean) может принимать только два значения: true (истина) и false (ложь).
