Базовые операторы, математика

Термины: «унарный», «бинарный», «операнд»

Терминология:

  • Операнд - то к чему приравнивается оператор.
  • Унарным называется оператор, который применяется к одному операнду. Например, оператор унарный минус "-" меняет знак числа на противоположный:
  • Бинарным называется оператор, который применяется к двум операндам. Тот же минус существует и в бинарной форме: x - y.

Математика

В JS существуют следущие математические операторы:

  • Сложение +,
  • Вычитание -,
  • Умножение *,
  • Деление /,
  • Взятие остатка от деления %,
  • Возведение в степень **.

Сложение строк при помощи бинарного +


let s = "моя" + "строка";
alert(s); // моястрока

alert( '1' + 2 ); // "12"
alert( 2 + '1' ); // "21"

alert(2 + 2 + '1' ); // будет "41", а не "221"

alert( 6 - '2' ); // 4, '2' приводится к числу
alert( '6' / '2' ); // 3, оба операнда приводятся к числам

Приведение к числу, унарный +

Используется для приведения к числу, например:


let apples = "2";
let oranges = "3";

// оба операнда предварительно преобразованы в числа
alert( +apples + +oranges ); // 5

Более читаемый вариант:


// alert( Number(apples) + Number(oranges) ); // 5

Сокращённая арифметика с присваиванием


//Арифметические 
testVar += 1   // сложение
testVar -= 1   // вычитание
testVar *= 1   // умножение
testVar /= 1   // деление
testVar %= 1   // остаток от деления
testVar **= 1  // возведение в степень

//Битовые 
testVar <<= 1  // левый сдвиг
testVar >>= 1  // правый сдвиг (со знаком)
testVar >>>= 1 // правый сдвиг (без знака)
testVar &= 1   // побитовое И
testVar |= 1   // побитовое ИЛИ
testVar ^= 1   // побитовое XOR

//Логические 
testVar &&= 1  // логическое И с присваиванием (ES2021)
testVar ||= 1  // логическое ИЛИ с присваиванием (ES2021)
testVar ??= 1  // нулевое присваивание (ES2021)

Арифметические используются достаточно часто, все остальное редко.

Инкремент/декремент

Существенные различия:

  • Когда оператор идёт после переменной — это «постфиксная форма»: counter++.
  • «Префиксная форма» — это когда оператор идёт перед переменной: ++counter.

Почему это важно:


let testVar = 2;
function test(x) {
    alert(x)
}
test(testVar++) //Тут 2 потому что сначала значение передаётся в ф-цию, потом инкремент значения.
test(++testVar) //А тут сначала инкремент, потом передача значения в ф-цию.

Побитовые операторы

  • AND(и) ( & )
  • OR(или) ( | )
  • XOR(побитовое исключающее или) ( ^ )
  • NOT(не) ( ~ )
  • LEFT SHIFT(левый сдвиг) ( << )
  • RIGHT SHIFT(правый сдвиг) ( >> )
  • ZERO-FILL RIGHT SHIFT(правый сдвиг с заполнением нулями) ( >>> )
