Преобразование типов

Библиотечные ф-ции автоматически переводят тип к нужному в своей работе. Однако этот вопрос надо изучить глубже, чтобы понять что и почему происходит.

Строковое преобразование

Общий паттерн состоит в:


let value = 1
let str = String(value)

Ф-ции по типу alert автоматически преобразуют входящие значения в String и выводят их на экран.
