Привет, мир!

Основные команды общения с миром, а так же браузер-специфичные вещи.

Тег «script»

Тег script может быть вставлен в любое место документа. Все функции которые "запущены" в скрипте будут выполнены при его загрузке. Загрузка осуществляется по мере загрузки документа.


<!DOCTYPE HTML>
<html>

<body>

  <p>Перед скриптом...</p>

  <script>
    alert( 'Привет, мир!' );
  </script>

  <p>...После скрипта.</p>

</body>

</html>

Современная разметка

По сути тут идёт перечисление атрибутов script которые не нужны:

  • type
  • langauge

Обёртывание скрипта в html-комментарии тоже не надо.
